# 🧠 Smart Prompt Templates

> Day 2 of my AI Engineering Journey — Built 5 reusable AI prompt templates.

## 📋 Templates Included
1. 📝 Summarizer — Summarizes any text into bullet points
2. ❓ Q&A — Answers any question clearly
3. 🔍 Explainer — Explains complex topics simply
4. 🌍 Translator — Translates text to any language
5. ✍️ Critic — Reviews and gives feedback on writing

## ⚙️ Tech Stack
| Tool | Purpose |
|------|---------|
| Python | Core language |
| Groq API | AI inference |
| LLaMA 3.3 70B | Language model |
| python-dotenv | API key management |

## 🚀 How to Run
1. Clone the repo
2. Install dependencies
```bash
   pip install groq python-dotenv
```
3. Add your Groq API key to `.env`
```bash
   GROQ_API_KEY=enter your API Key
```
4. Run
```bash
   python main.py
```
